dead by Daylight

vc sabe oq é dbd(Dead by Daylight)?
Dead by Daylight é um jogo de terror de sobrevivência online para várias pessoas. Um jogador assume o papel de um assassino brutal, e os outros quatro jogadores são os sobreviventes.O objetivo do assassino é caçar e sacrificar todos os sobreviventes em ganchos para agradar uma entidade maligna conhecida como a Entidade. Já os sobreviventes, precisam trabalhar juntos para consertar geradores e abrir portões de saída para fugir do local. Se um sobrevivente for pego, ele tem que lutar para se libertar ou ser resgatado por outro jogador.

Projeto deselvolvido por Heloiza, sem fins lugrativos.
